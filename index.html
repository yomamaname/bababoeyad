Free hosting path

Ready for GitHub Pages

This site is now a Jekyll project with a GitHub Actions workflow that builds the static site and publishes it to GitHub Pages. Push to main, enable Pages with “GitHub Actions” as the source, and the potato garden can be hosted for small audiences at no server cost.

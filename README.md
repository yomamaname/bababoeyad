# Potato Reverie

A Jekyll-powered website celebrating the beauty of potatoes and recipes made from them.

## Run locally

```bash
bundle install
bundle exec jekyll serve
```

Visit `http://127.0.0.1:4000`.

## Free hosting with GitHub Pages

This repository includes `.github/workflows/pages.yml`, which builds the Jekyll site and deploys it to GitHub Pages whenever `main` is pushed.

1. Push this repository to GitHub.
2. In repository settings, open **Pages**.
3. Set **Source** to **GitHub Actions**.
4. Push to `main` or run the workflow manually.

GitHub Pages is a good free option for a static Jekyll website intended for a small number of users.
